Modelo de api em NodeJS com MongoDB

### Pré-requisitos
- node
- yarn

### Instalação
O uso do yarn pode ser substituído pelo npm.
```bash
yarn
```

### Rodando o projeto
Antes de rodar o projeto, é necessário criar um arquivo ```.env.development``` com os atributos do ```.env.example```.
```bash
yarn start
```
